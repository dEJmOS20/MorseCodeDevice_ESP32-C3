# MorseCodeDevice_ESP32-C3
Simple code for the microcontroller, casing design for printing, and electronics used in the project. The project was created during the holidays (a 4-day run), many things could have been done better and simpler (especially in the casing and practical execution). I invite you to take inspiration, improve, and seek ideas &lt;3

3d models on my profile: 

This device is designed primarily for learning Morse code rather than long-distance communication - though with a few modifications, it can serve that purpose as well. It features an audible signal and a real-time decoder, so you can check your accuracy and track your progress as you practice.
The electronics are straightforward to connect. It requires just an OLED display, two pull-up buttons, and a buzzer to get up and running (the rest is as an addition). Schematic will be added in the future.
