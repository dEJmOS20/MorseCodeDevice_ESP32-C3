# MorseCodeDevice_ESP32-C3 // PlatforIO; Arduino code
Included in this repository: the microcontroller code, a 3D-printable enclosure design, and the electronics used in the project.
The project was built over a 4-day stretch during the holidays — quick and fun, though there's plenty of room for improvement, particularly in the enclosure design and overall execution. Feel free to take inspiration, build upon it, or use it as a starting point for your own ideas. ❤️

3d models on my profile: https://www.thingiverse.com/thing:7330210
Video: https://www.youtube.com/shorts/dO_yk2Q_z7g

This device is designed primarily for learning Morse code rather than long-distance communication - though with a few modifications, it can serve that purpose as well. It features an audible signal and a real-time decoder, so you can check your accuracy and track your progress as you practice.
The electronics are straightforward to connect (tips in code). It requires just an OLED display, two pull-up buttons, and a buzzer to get up and running (the rest is as an addition). Schematic will be added in the future.
